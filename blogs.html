How To Install Arch Linux

2022 April 3

This document is a guide for installing Arch Linux using the live system booted from an installation medium made from an official installation image. For more detailed instructions, see the respective ArchWiki. Arch Linux should run on any x86_64-compatible machine with a minimum of 512 MiB RAM, though more memory is needed to boot the live system for installation. A basic installation should take less than 2 GiB of disk space. You need a working internet connection throughout this process.
